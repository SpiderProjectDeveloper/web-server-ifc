`Creating a Project:
Choose an .ifc file. After it is loaded it is visualized in the middle area of the browser window (the 3D View) and an hierarchy tree is displayed at it's right area (the IFC Tree View).
Create an hierarchical structure (phases and operations) of your project (the Activities Hierarchy View that placed in the left area of the window). To add/remove elements use the right mouse button to call the context menu. You may as well move elements across the hierarchy with your mouse. Make a double click to edit the name of a phase or operation element in the hierarchy. When clicked an element is highlighted. The highlighted operation can be linked with elements of your 3D model. You have several ways to make linking:
* pick an element in the 3D view, or
* Check a checkbox in the IFC Tree View, or
* Use the Project Setup Tool (${this.icons.attach} button).
To make picking of elements easier you may hide some unnecessary elements leaving at the 3D View only a subset of those. Right click an element in the IFC Tree View and check all child elements of a required type.
After all the elements required are linked press the save button ${this.icons.save} to write down the changes into the Spider Project App.

Project Setup Tool
First you must create a list of works each having the name, type and color code - the Works Types window is available by clicking the ${this.icons.works} button. Next for some or every work type you point out a set of IFC-elements and related materials (please use the ▷ button). By clicking the ${this.icons.attach} button each storey of the IFC-model would be automatically parsed and a set of related operations would be created.` `Создание проекта:
Выберите .ifc файл. Когда он загрузится в средней части окна браузера появится визуализация модели (Окно 3D), а в правой части - его иерархическая структура (Окно IFC-структуры).
Создайте структуру проекта (фазы и операции) в левой части окна бразуера (Окно структуры проекта). При загрузке приложения структура проекта представлена единственным корневым элементом, который по умолчанию имеет название "Проект". Для добавления/удаления новых элементов (фаз и операций) используйте контекстное меню, которое вызывается по нажатию правой кнопки мыши на каждом элементе структуры. проекта. Созданные элементы можно перетаскивать с места на место внутри структуры. Редактировать имя элемента можно двойным щелчком.
Щелчок мышью по элементу подсвечивает его. К подсвеченному элементу может быть привязаны элементы 3D-модели. Привязку можно сделать несколькими способами:
* Щелкнув мышью по изображению элемента в Окне 3D.
* Поставив галочку напротив элемента в Окне IFC-структуры.
* Воспользовавшись инструментом для автоматического создания операций.
Чтобы сделать привязку элементов по щелчку в Окне 3D более удобной, можно оставить видимыми на экране лишь часть элементов модели. Для этого надо щелкнуть по элементу в Окне IFC-структуры правой кнопкой и в открывшемся меню выбрать нужный IFC-тип - на экране останутся только элементы этого типа, дочерние по отношению к элементу, для которого было вызвано меню.
После того, как все нужные элементы будут привязаны к операциям, нажмите кнопку ${this.icons.save}, и данные будут записаны в приложение Спайдер Проджект.

Автоматическое создание операций
Сперва вам необходимо задать список работ, указав для каждой имя, тип и цветовой код. Это делается в окне Кодировка типов работ, доступном по нажатию кнопки ${this.icons.works}. По нажатию кнопки ▷, расположенной слева от каждого типа работ, открывается панель, где можно выбрать IFC-элементы и связанные с ними материалы, которые будут "прикреплены" к автоматически созданным операциям после нажатия кнопки ${this.icons.attach}.`
